# Databases MCP (Archived)

Central inventory and knowledge base for network operations. 

Note: This component is archived and preserved for reference.

## Tools

- `db_olt_list()`: List all registered OLTs.
- `db_olt_save()`: Add or update OLT credentials.
- `db_olt_delete()`: Remove an OLT from inventory.
- `db_olt_cmds_list()`: Search CLI command syntax.
- `db_ont_list()`: List available ONT batches.
- `db_ont_summary()`: Get high-level stats for ONT batch.
- `db_ont_clusters()`: Identify interference clusters.
- `db_ont_details()`: Get cluster details.
- `db_ont_status()`: Get ONT radio status.
- `db_skills_list()`: List available SOP categories.
- `db_skills_read()`: Read full SOP content.
